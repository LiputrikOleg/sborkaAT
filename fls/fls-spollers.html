Спойлеры

Старая версия отличается от новой. Старый HTML-код работает со старым Javascript.

Начало

HTML

Сниппет : spollers

<div data-spollers class="spollers">
  <details class="spollers__item">
    <summary class="spollers__title">Заголовок спойлера</summary>
    <div class="spollers__body">Контент спойлера</div>
  </details>
</div>
Описание

Основные атрибуты споллера

Для родителя слойлеров пишем атрибут data-spollers

Если нужно включать\выключать работу спойлеров на разных размерах экранов пишем параметры ширины и типа брейкпоинта.

Например:

data-spollers="992,max" - спойлеры будут работать только на экранах меньше или равно 992px

data-spollers="768,min" - спойлеры будут работать только на экранах больше или равно 768px

Оформление

Базовые стили

// Блок со спойлерами
.spollers {
  // display: grid;
  // gap: 15px;
}

// Спойлер
.spollers__item {
  font-size: toRem(20);
  margin-bottom: 30px;
}

.spollers__body {
  padding: 1.25rem;
  border-radius: 0.3125rem;
  border: 1px solid #eee;
}

Декоративные штучки типа : крестики, уголки - перевертыши, стрелочки и др. появятся только тогда, когда у класса родителя появится технический класс _spoller-init

      right: toRem(7);
    }

  }

      transform: rotate(-40deg);
    }

    }
  }
