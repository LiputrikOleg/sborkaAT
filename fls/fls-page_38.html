Основные возможности

Красиво раскрашивает выводимый в тегах <pre><code>....</code></pre> код.

Подключение функционала

Из скаченного архива используем только папку styles

Из неё выбираем необходимый файл CSS и копируем его либо в файл src/scss/libs/highlight/dark.scss, либо в src/scss/libs/highlight/light.scss/, в зависимости от темы.

[HTML] В нужном месте документа в тег <pre><code>....</code></pre> вставляем код HTMl, CSS, JAVASCRIPT, PHP

[JS] В файле src/html/_js.htm дописываем:

<script src="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.4.0/highlight.min.js"></script>
<script>hljs.highlightAll();</script>

Тем самым подключаем JS-файл из интернета и инициализируем его

[SCSS] в файле src/scss/ style.scss расскоментируем необходимую строку:

// Цветовая схема кода
// @import "libs/highlight/dark.scss";
// @import "libs/highlight/light.scss";

Подключение без интернета

Открываем скрипт .

В папке src/js/files создаем файл highlight.js.

В него помещаем содержимое выше указанного скрипта. Просто содержимое, без всяких тегов <script> </script>

Перед содержимом скрипта прописываем:

import { flsHighlight } from "./highlight.js";

Точно такую же запись прописываем в файле src/js/files/script.js

Возможно нужно будет просто раскомментировать данную строку

// Подключение подсветки кода
import { flsHighlight } from "./highlight.js";

Возвращаемся в файл src/js/files/highlight.js и после кода скрипта инициируем скрипт

hljs.highlightAll();

без всяких тегов <script> </script>
