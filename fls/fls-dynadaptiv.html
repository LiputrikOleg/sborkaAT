Динамический адаптив (Dynamic Adapt)

Это JS функция для комфортной адаптивной верстки.

Позволяет "перебрасывать" объекты DOM в зависимости от потребностей.

Применение

Для перемещаемого объекта пишем HTML атрибут - `data-da` и указываем параметры. 

data-da = "куда,когда,какой"`

Параметры

Параметр Описание
`куда (имя класса)`
| обязательно |		 Класс блока, в который нужно будет "перебросить" текущий объект. Если класс не уникален, объект перебросится в первый элемент с этим классом.
`когда` | 767 | Брейкпоинт, при котором перемещается объект.
`какой` | last | Позиция на которую нужно переместить объект внутри родителя `куда`. Кроме цифр можно указать слова `first` (в начало блока) или `last` (в конец блока)

По умолчанию, если 'data-da' задан, `куда (имя класса)` указан, а параметры `когда` и `какой` не указаны - Объект переместится в блок с указанным классом при ширине экрана 767px и займет последнее место ( станет самым нижним в блоке).

Порядок ( позиция ) следования объектов в "принимаемом блоке" 0, 1, 2, 3... и это надо учитывать при указании `какой`

JavaScript

Функционал скрипта для работы расположен "src ==> js ==> libs ==> dynamic_adapt.js"

В javaScript создан объект класса DynamicAdapt ( В самом конце ) с параметром "min" или "max"

Тип срабатывания брейкпоинта:

max - Desktop First  равносильно  @media (max-width: Npx)
min - Mobile First  равносильно   @media (min-width: Npx)

const da = new DynamicAdapt("max");
    da.init();

Примеры

Я Город

Я Коля
Я Аня
Я Ваня

Я бабушка

Я огород

Я Лоцы

Я речка

Я Оля

Упрощенный HTML ( Классы оформления опущены )

<section class="section">
<h2>Я Город</h2>
<div class="blocks-city">
  <div data-da=".column-garden">Я Коля</div>
  <div data-da=".column-garden, 992, 1">Я Аня</div>
  <div>Я Ваня</div>
</div>
</section>

<section class="section">
<h2>Я бабушка</h2>

<div class="row">
  
  <div class="column-garden">
  <h3>Я огород</h3>
  <div data-da=".column-river, 480, 1">Я Лоцы</div>
  </div>

  <div class="column-river">
    <h3>Я речка</h3>
    <div data-da=".blocks-city, 650, 0">Я Оля</div>
  </div>

</div>
</section>

Описание действий

  1. При ширине 992рх Аня переместится в огород на первую позицию ( "Я огород" - позиция 0 )
  2. При ширине 767рх Коля переместится в конец огорода ( По умолчанию )
  3. При ширине 650рх Оля переместится в город на нулевую позицию
  4. При ширине 480рх Лоцы переместится на речку на первую позицию ( "Я речка" - позиция 0 )
