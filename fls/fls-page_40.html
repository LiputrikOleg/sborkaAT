Подключение дополнительных модулей (Эксперимент)

Подключение без интернета

На примере подключения подсветки кода

[HTML] Пишем необходимый код или пользуемся сниппетом, если он есть.

Скачиваем или копируем скрипт (например скрипт ).

[JS] В папке src/js/files создаем файл highlight.js.

В него помещаем содержимое выше указанного скрипта. Просто содержимое, без всяких тегов <script> </script>

Перед содержимом скрипта прописываем:

import { flsHighlight } from "./highlight.js";

Точно такую же запись прописываем в файле src/js/files/script.js

Возможно нужно будет просто раскомментировать данную строку

// Подключение подсветки кода
import { flsHighlight } from "./highlight.js";

Возвращаемся в файл src/js/files/highlight.js и после кода скрипта инициируем скрипт

hljs.highlightAll();

без всяких тегов <script> </script>

[SCSS] В папке src/scss/libs/ создаем файл со стилями

в файле src/scss/ style.scss подключим необходимую строку:

// Цветовая схема кода
// @import "libs/highlight/dark.scss";
// @import "libs/highlight/light.scss";

Другой вариант:
В папке src/scss/base/ создаем файл со стилями

Теперь в файле src/scss/base.sccs пропишем 

// Стили этого модуля
@import "base/файл без расширения";
( например: @import "base/btn-up";)
