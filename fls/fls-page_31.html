Модуль «Ripple Effect»

Оглавление
Подключение Будет полезно

Подключение

Модуль добавляет к любому объекту эффект волны при щелчке

Подключение

[HTML] К любому объекту добавьте дату атрибут data-ripple 

<button type="submit" data-ripple class="button">Отправить</button

[SCSS] В файле src/scss/base.scss раскомментируйте строку @import «base/ripple»; Файл src/scss/base/ripple.scss содержит нужные стили для эффекта. Эти стили можно изменять под себя, например время анимации: 

.ripple {
/* ... код ... */
animation: button-circle 1.5s ease forwards;
}

[JS] в файле src/js/app.js раскомментируйте строку flsFunctions.rippleEffect();
