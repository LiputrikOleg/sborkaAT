Параллакс-эффект при скролле страницы

Подключение модуля

В шаблоне встроен функционал параллакс-эффекта при скроллах страницы. Авторы Хмурый Кот, Евгений Андриканыч.

[JS] В файле js/app.js раскоментировать строку import './libs/parallax.js';

[HTML] Строим HTML-структуру с добавлением атрибутов data-prlx-parent для родительского элемента и data-prlx для элементов, для которых хотим применить эффект. 

<div data-prlx-parent class="page">
  <div data-prlx class="page__image"></div>
</div>
