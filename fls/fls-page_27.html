Подключение модуля

В шаблоне встроен функционал поэкранной прокрутки (скролла) страницы. Авторы Хмурый Кот, Фрилансер по жизни.

[JS] В файле js/app.js раскоментировать строку @import‘./libs/fullpage.js’;

[SCSS] В файле src/scss/base.scss раскомментировать строку @import «base/fullpage»;

[HTML] Использовать сниппет fillpage, либо построить HTML-структуру самостоятельно

<div data-fp class="some-class">
  <div data-fp-section class="some-class__section"></div>
...
</div>

Настройки модуля

Булеты

Существует возможность добавить кнопки переключения секций (буллеты). Для этого следует к объекту с дата атрибутом data-fp добавить дата атрибут data-fp-bullets 

<div data-fp data-fp-bullets class="some-class">
  <div data-fp-section class="some-class__section"></div>
...
</div>

Эффекты переключения

Для смены эффекта переключения, объекту с дата атрибутом data-fp добавить дата атрибут data-fp-effect и в качестве значения добавить один из следующих параметров fade, cards, slider (по умолчанию) 

<div data-fp data-fp-effect="fade" class="some-class">
  <div data-fp-section class="some-class__section"></div>
...
</div>
