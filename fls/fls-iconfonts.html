Создание собственного иконочного шрифта из SVG-файлов

Из архива

Выбираем необходимые иконки из скаченного архива.

Папка "svgs" ==> Нужная категория ==> иконка(и).

Скопируем их в папку проекта "iconsfont".

Платные иконки

Выбираем платную иконку, кликаем по ней, в открывшимся окне выбираем любую миниатюру (размер не имеет значения) ==> "Просмотреть код элемента" ==> копируем тег svg.

В папке проектка создаем новый файл .txt и вставляем туда код.

Сохраняем в папку iconfonts c любым именем и расширением .svg. Обычно имя берут с сайта иконок.

Иконки с макета

Кликаем иконку и cохраняем в папку iconfonts c раширением .svg.

Для того, чтобы не выскакивала ошибка при загрузкке НУЖНО:

  1. В Figme кликнуть по иконке.
  2. Нажать комбинацию клавиш Chift + Ctrl +O.
  3. Выбрать расширение ( svg ).
  4. Экспортировать в папку назначения.
  5. Загрузить на IcoMoon.
  6. Если выскакивает окошко с ошибкой, перейти по ссылке посмотреть решение проблемы.

Переходим на сайт IcoMoon.

Кликаем на фиолетовую кнопку "IcoMoonApp".

Menu (слева бергер) ==> New Empty Set.

Untitled Set ==> справа бургер-menu ==> Import To Set ==> Выбираем все иконки.

Делаем иконки черно-белыми - клик на набор цветов ==> корзина.

Меню слева от поиска:

  1. Стрелка - выбираем иконки, которые добавим в шрифт.
  2. Корзина - удаляем ненужные.
  3. Перекрестье - перетаскивание иконок ( зачем? ).
  4. Карандаш - редактирование.

Выделяем иконки и кликаем (внизу справа) Generate Font.

Появятся иконки. т.к.их имена преобразуются в классы при верстке есть смысл их сейчас уменьшить - отредактировать.

После редактирования нажимаем шестеренку (внизу справа за загрузкой).

1. Название шрифта, под которым сохранятся иконки.

2. Префикс, который будет добавлятся к классу иконок.

Настройки можно менять при неоходимости, но лучше оставить по-умолчанию.

"Download"

Скачивается архив.

Папку "fonts" и файл "style.css" переносим в проект.

Файл "style.css" переименовываем в "iconsfont.css" и переносим в папку CSS.

Подключаем файл iconfonts.css до основного файла стилей.

Возможно, будет необходимо в этом файле изменить путь подключения папки fonts.

Также добавляем псевдоэлемент ::before 14:20.

Подключение в HTML

Просто добавляем класс иконки.

<a href="#" class="link _icon-jack">Фрилансер по жизни</a>

В Gulp-сборке

В папке src создаем папку iconfonts и загружаем туда иконки с макета с расширением .svg.

В папке src создаем папку fonts и загружаем туда из скаченого архива icomoon-v1.0.zip ==> папки fonts ==> файл icons.ttf.

Из файла style.css архива выбираем стили с классами иконок.

И переносим их в src ==> scss ==> fonts ==> icons.scss ниже расположенной там записи.

Возможно потребуется изменить font-family в самом верху файла.

Gulp сделает необходимую работу и нам остается в файле src ==> scss ==> style.scss расскомменировать строчку:

@import "fonts/icons";
