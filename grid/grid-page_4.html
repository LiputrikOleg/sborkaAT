Позиционирование (координаты) элементов

grid-column-start - Указывает grid-элементу на какой линии колонок он начинается

grid-column-end - Указывает grid-элементу на какой линии колонок он заканчивается

grid-row-start - Указывает grid-элементу на какой линии рядов он начинается

grid-row-end - Указывает grid-элементу на какой линии рядов он заканчивается

При наличии большого количества колонок и рядов возможно использование обратного счета

Равнозначная запись

.block__grid {
   display:grid;
   grid-template-columns: repeat(5, 1fr);
   grid-template-rows: repeat(3, 1fr);
   }
   .item2 {
   grid-column-start: -5;
      grid-column-end: -2;
      grid-row-start: -4;
      grid-row-end: -2;
   }

Сокращенная запись

grid-column: grid-column-start / grid-column-end;
      grid-row: grid-row-start / grid-row-end;
      
.item2 {
grid-column: 2 / 5;
grid-row: 1 / 3;
}

А теперь пример с указанием всех четырёх значений в строку. При таком указании значений есть скользкое место: значения указываются в таком порядке: row-start / column-start / row-end / column-end, то есть сначала указываем оба начала, а потом оба конца. 


 grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end;
           
.item2 {
grid-area: 1 / 2 / 3 / 5;
}

С помощью специального слова span можно задать растяжение элемента на несколько ячеек.
После слова span указывается, на какое количество ячеек надо растянуть элемент:

Расшифровка записи ниже:
блок начинается на 1 линии колонок и растягивается на 4 колонки
начинается на 2 линии рядов и растягивается на 3 ячейки. 

.grid-block {
display: grid;
grid-template-columns: repeat(5, 1fr);
grid-template-rows: repeat(5, 30px);
}
.item_1 {
grid-column-start: 1;
grid-column-end: span 4;
grid-row-start: 2;
grid-row-end: span 3;
}

Создаем grid-сетку и "компануем" интернет магазин. Код записан так, что понятна "компановка" страницы

CSS 
.cart-product {
display: grid;
grid-template-columns: repeat(6, 1fr);
/* 6 столбцов одинаковой ширины для десктопа */
grid-template-rows: repeat(5, 1fr);
/* 5 строки автоматической высоты */
grid-gap: 10px;
/* промежуток между блоками */
grid-template-areas:
   "image       image       image         title        title    title"
   "image       image       image         price        rating   rating"
   "features    features    features      price        payment  payment"
   "description description description   description  reviews  reviews  "
   "shipping    contact     availability  availability warranty sizes"
   "material    material    related       related      related  related";
}

Что-бы все это работало нужно каждому из блоков присвоить персональное "имя". ( стили оформления: размеры, цвета и др. опущены.)

Этот способ очень удобен при компоновке блоков различных размеров. Не требуется расчета, где будет стоять и сколько места занимать каждый из блоков. В коде визуально понятно, где, кто и какой.

Для различных размеров экрана нужно лишь скорректировать код.

CSS 
@media (max-width: 720px) {
   .cart-product {
      grid-template-columns: repeat(4, 1fr);
      /* 4 столбца одинаковой ширины для мобильника */
      grid-template-areas:
      "image        image    price       rating"
      "image        image    description description"
      "title        title    title       title"
      "reviews      reviews  features    features"
      "availability payment  shipping    contact"
      "warranty     warranty sizes       sizes"
      "warranty     warranty sizes       sizes"
      "material     material material    related";
   }
}
