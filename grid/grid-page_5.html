Направление и порядок элементов

Применяется для Grid-контейнера


По умолчанию все элементы располагаются по порядку горизонтально, если места в строке больше нет, то элементы переносятся на следующую строку.
Но с помощью свойства grid-auto-flow можно изменить направление элементов. Это свойство принимает два значения:


  1. row: значение по умолчанию, элементы располагаются в строку друг за другом, если места в строке не хватает, элементы переносятся на следующую строку

  2. column: элементы располагаются в столбик, если места в столбце не хватает, то элементы переходят в следующий столбец

Grid-элементы располагаются в обычном порядке - сначала первый ряд, затем второй и т.д.

.grid-block {
   display: grid;
   grid-template-columns: repeat(3, 150px);
   grid-template-rows: repeat(3, 150px);
   grid-auto-flow: row;
}

Grid-элементы должны располагаться порядно, т.к. grid-auto-flow: row; , но браузер видит, что в отведенное для блока 3 место - справа от блока 2 блок 3 не помещается т.к. имеет ширину в две ячейки - grid-column: span 2; , поэтому он переносит блок 3 на нижний ряд, отодвигает блок 4 вправо на свободное место (третья колонка). Чтобы этого не происходило применяется замещение. Смотри следующий TAB.

.grid-block {
   display: grid;
   grid-template-columns: repeat(3, 150px);
   grid-template-rows: repeat(3, 150px);
   grid-auto-flow: row;
   }
   
   .item3 {
   grid-column: span 2;
}

Для автозамещения служит значение dense для свойства grid-auto-flow: и теперь, если элемент 3 не помещается в ячейку справа от элемента 2, то браузер вставляет туда следующий за третьим элементом блок. Третий переносит в нижний ряд, и т.к. четвертый блок уже пристроен - следующим ставит блок 5, затем блок 6. Получается тот вид, который видим. 

.grid-block {
   display: grid;
   grid-template-columns: repeat(3, 150px);
   grid-template-rows: repeat(3, 150px);
   grid-auto-flow: row dense;
   }

   .item3 {
   grid-column: span 2;
}

Т.к все элементы Grid-контейнера по-умолчанию имеют order:0; , то значение order:-1; ставит элемент первым, значение order:1; - последним. Изменяя можно переставлять Grid_элементы в любом порядке. Допустимы любые целые числа. 

.grid-block {
   display: grid;
   grid-template-columns: repeat(3, 150px);
   grid-template-rows: repeat(3, 150px);
   grid-auto-flow: row dense;
}
     
   .item3 {
   grid-column: span 2;
   order:-1;
}
