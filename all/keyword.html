Горячие сочетания клавиш

Горячие клавиши Visual Studio Code

  • Запуск терминала (строка состояния) - ctrl + J
  • Запуск терминала (строка состояния) - ctrl + `

  • Поиск кода по номеру строки - ctrl + G

  • Перемещение выделеного фрагмента на одну строку вверх или вниз - ctrl + ↑ или ctrl + ↓

  • Множественный курсор - ctrl + клик
  • Множественный курсор для выделенного слова - все вхождения - ctrl + F2

  • Дублирование строки или выделения ниже - shift + alt + ↓
  • Дублирование строки или выделения выше - shift + alt + ↑

  • BEM создание элемента - alt + B  E
  • BEM создание элемента + модификатора - alt + B  M

  • Стили для CSS - ctrl + alt + 1
  • Стили для SCSS - ctrl + alt + 2
  • Стили для SCSS с комментариями - ctrl + alt + 3

  • Переводчик с выбором языка - ctrl + alt + Y
  • Переводчик русско-английский - ctrl + alt + T

  • Увеличение шрифта редактора - ctrl + +
  • Уменьшение шрифта редактора - ctrl + -

    • или

  • Изменение шрифта редактора- ctrl + колесико мыши

  • Выделить строку - ctrl + L - повторное нажатие комбинации выделяет строку ниже
  • Обернуть тегом - ctrl + shift + A

  • Переход между вкладками - ctrl + Z

  • Просмотреть картинку - ctrl + левая кнопка мыши - клик по ссылке img

  • JS - замена переменных или констант - выделить -> F2 - новое имя (замена по всему коду, где используется переменная)
  • Js - просмотреть, где используется переменная - выделить -> F12
