В окошке PUBLISH отображается количество выбранных иконок, по окончании выбора кликаем по ней. Существуют 2 варианта отображения иконок: Icon Font или SVG Sprite. Выбираем нужный и Download - скачиваем

Для каждой иконки существует свой код. В таблице стилей смотрим его – например для лупы это \61. Чтобы иконку вставить в поле поиска сделаем так:

<input type="submit" value="&#x61;">

Из той же таблицы стилей копируем шрифт ( font-family: "myfont" !important; ) и прописываем его этой кнопке в основных стилях. Теперь у нас выводиться лупа в кнопке submit.

Icomoon

  • Заходим на сайт https://icomoon.io
  • Кликаем вверху кнопочку IcoMoon App
  • Выбираем необходимые иконки
  • Внизу кнопка Generate Font - кликаем
  • Download
  • Создаем новую папку iconmoon, в нее из скаченного архива переносим файл style.css и папку fonts
  • Подключаем 
    <link rel="stylesheet" href="iconmoon/style.css">
  • В этом файле указаны классы иконок.
  • Чтобы вывести иконку на экран подключаем ее через пустой тег <i> 
    <div class="p2__bullet"> <i class="icon-star-full"></i>7 сайтов своими руками</div>
  • Иконка появляется непосредственно перед текстом, чтобы переместить иконку влево на некоторое расстояние применяем позиционирование:
  • Родительскому блоку <div class="p2__bullet"> задаем position: relative;
  • Иконкам position: absolute; top:0; left: 5px; или другой сдвиг
  • Текст соответственно сдвигаем вправо при помощи padding-left: 70px;
  • Иконки это фактически текст, поэтому стили как для текста font-size: 23px;color: #f4924a;font-weight: 400;
