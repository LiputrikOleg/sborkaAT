Сематические теги

<main> - В нем должен содержаться главный контент страницы

Важно:

1. Этот тег должен быть уникальным и встречаться один раз на странице.

2. Он не должен быть внутри других структурных тегов.

<header> - Шапка сайта,используется в статьях для показа названия статьи. Так же используется как заголовочная информация.

<nav> - Главная навигация (меню) по сайту

<article> - Контент, основная информация сайта (статьи).

<section> - В этот тег заключают либо каждую статью, либо части статьи, либо какой то раздел (для сравнения главы, части, предисловия и т.д в книге)

<hgroup> - Объединяет все теги Н1-Н6. При группировке влияние будет иметь только самый первый тег, а остальные будут использоваться в качестве описания

<aside> - Используется в основном в сайдбарах. Вторичная информация.

<footer> - Подвал сайта

Важно:

Для полноценного использования тегов HTML5 во всех браузерах достаточно выполнить три условия:

1.Установить доктайп

<!DOCTYPE html>

2.Включить скрипт, расположеный ниже

3.В стилях для новых тегов установить display: block; , в противном случае стили не будут работать.
