Адаптация изображений

img {
  max-width: 100%;
  height: auto;
}

Для того, чтобы на разных расширениях экрана загружались различные по формату рисунки:

<picture>
  <source media="(max-width: 480px)"
  srcset="img/mobile.jpg">               // до ширины 480px загружается mobile.jpg
  <source media="(max-width: 640px)"
  srcset="img/tablet.jpg">               //от 480рх до 640рх  tablet.jpg 
  <img src="img/img-full.jpg" alt="">      // далее img-full.jpg

  <img src="img/img-800.jpg"                // Для DPR:1.0
  srcset="
  img/img-800.jpg 800w,                     // Для DPR:2.0
  img/img-full.jpg 1400w"                   // Для DPR:3.0
  alt="">
</picture>

* DPR это соотношение виртуальных пикселей к физическим

Для браузера способного воспроизвести изображение с разрешением .webp загружается файл img-full.webp , а для неспособных - загружается img-full.jpg

<picture>
  <source type="image/webp" srcset="iimg-full.webp">
  <img src="iimg-full.jpg" alt=""  
</picture>

<video class="video" width="768" height="432" controls src="../../video/clip.mp4">
  <sourse src="../../video/clip.mp4" type="video/mp4"></sourse>
</video>

.video {
  width: 100%;
  height: auto;
  display: block;
  /* можем задать фиксированную ширину,тогда адаптация пойдет после 480рх.*/
  max-width: 480px;
  margin: auto;
}
