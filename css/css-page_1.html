Выравнивание

text-align - Выравнивание текста

  • text-align: center ; /* Выравнивание текста по центру */
  • text-align: justify; /* Выравнивание по ширине, что означает одновременное выравнивание по левому и правому краю */
  • text-align: left; /* Выравнивание текста по левому краю */
  • text-align: right; /* Выравнивание текста по правому краю */
  • text-align: auto; /* Не изменяет положение элемента */
  • text-align: start; /* Аналогично значению left, если текст идёт слева направо и right, когда текст идёт справа налево */
  • text-align: end; /*Аналогично значению right, если текст идёт слева направо и left, когда текст идёт справа налево */
  • text-align: inherit; /* Наследует значение родителя */

text-align-last - Задает выравнивание последней строки текста, когда свойство text-align установлено как justify.

  • text-align-last: center /* - Строка выравнивается по центру*/
  • text-align-last: justify /* - Строка выравнивается по ширине. Если в последней строке только одно слово, то оно будет выравнено по левому краю */
  • text-align-last: left /* - Строка выравнивается по левому краю*/
  • text-align-last: right /* - Строка выравнивается по правому краю*/
  • text-align-last: start /* -Строка выравнивается по начальному краю блока */
  • text-align-last: end /* - Строка выравнивается по конечному краю блока*/

Красная строка

text-indent - Устанавливает величину отступа первой строки блока текста (например, для абзаца <p>). Воздействия на все остальные строки не оказывается. Допускается отрицательное значение для создания выступа первой строки, но следует проверить, чтобы текст не выходил за пределы окна браузера. В качестве значений принимаются любые единицы длины.

Преобразование текста

text-transform

  • text-transform: capitalize; - Первый символ каждого слова в предложении будет заглавным. Остальные символы свой вид не меняют
  • text-transform: lowercase; - Все символы текста становятся строчными (нижний регистр)
  • text-transform: uppercase; - Все символы текста становятся прописными (верхний регистр)
  • text-transform: none; - Не меняет регистр символов
  • text-transform: inherit; - Наследует значение родителя

Переносы слов

Переносы слов

text-overflow - Определяет параметры видимости текста в блоке, если текст целиком не помещается в заданную область. Возможны два варианта: текст обрезается; текст обрезается и к концу строки добавляется многоточие

  • clip - обрезает текст, который не помещается в блоке.
  • ellipsis - делает тоже самое, только добавляет в конце троеточее.
<div class="texo">
Это обычный текст, который мы будем использовать на этой странице, в этом абзаце. Вот так !!!! Lorem ipsum dolor sit amet,
consectetur adipisicing elit. Eaque nobis, adipisci dicta consectetur! Iure, inventore quaerat deserunt, eaque ullam doloremque.
</div>
.texo {background: #ffd680; text-overflow:ellipsis;white-space: nowrap; overflow: hidden;}
Это обычный текст, который мы будем использовать на этой странице,в этом абзаце. Вот так !!!! Lorem ipsum dolor sit amet, consectetur adipisicing elit. Eaque nobis, adipisci dicta consectetur! Iure, inventore quaerat deserunt, eaque ullam doloremque.

Важно: Это свойство работает, только если свойство overflow имеет значения auto, scroll и hidden

word-break - указывает как следует переносить длинные слова.

  • normal - Используются правила переноса строк по умолчанию. Как правило, в этом случае строки не переносятся или переносятся в тех местах, где явно задан перенос (например, с помощью тега <br>).
  • break-all - Перенос строк добавляется автоматически, чтобы слово поместилось в заданную ширину блока. Значение не работает для текста на китайском, корейском или японском языке.
  • keer-all - Не разрешает перенос строк в словах на китайском, корейском или японском языке. Для остальных языков действует как normal.
<div class="wb">
Этомынаписалицеликомбезпробеловдляэкспериментакоторыймысейчасбудемставить.</div>

.wb {background: #ffd680;width: 200px;word-break: break-all;}

Тег <wbr> - указывает браузеру, где необходимо разместить перенос.

<div class="wbr">Это &shy; мынаписалицеликом <wbr>безпробелов<wbr>дляэксперимента<wbr>которыймысейчасбу &shy; демставить.</div>
Это­мынаписалицеликом безпробеловдляэкспериментакоторыймысейчасбу­демставить.

Важно: Тег <wbr> и мягкий перенос слов не работают, если указано свойство word-break:break-all;.

Важно: Ни один браузер при переносе текста не добавляет символ дефиса. При необходимости дефис можно имитировать, заменив тег <wbr> на символ мягкого переноса &shy;. Этот символ должен стоять в конце строки, иначе перенос игнорируется (смотри предыдущий пример).

Internet Explorer 8 и Opera до версии 11.7 не делают переносы длинной строки, пока в стилях для <wbr> не указано свойство display со значением inline-block, как показано в примере. 

wbr {display:inline-block}

Свойство hyphens

И, наконец, самое мощное и удобное свойство для автоматического добавления переносов — hyphens. Его действие основано на встроенном в браузере словаре переносов, поэтому даёт наилучший результат. Поддерживается в IE10, Firefox, Android и iOS. Chrome и Opera не поддерживают. Чтобы это всё работало, для тега <html>добавляем атрибут lang со значением ru.

Запрет переносов

Нередко возникает и обратная задача — запретить переносы в тех местах, где они по правилам языка недопустимы. Например, нельзя отрывать единицы измерения от числа (10 мл), обозначение года (54 до н. э.), инициалы от фамилии, разрывать устойчивые сокращения (т. д.) и др. Чтобы браузер не добавлял переносы в месте пробела, его следует заменить на неразрывный пробел &nbsp; .

Озеро по координатам 70° 58′ 19″ с. ш. 97° 24′ 5″ в. д. расположено в Таймырском Долгано-Ненецком районе Красноярского края России.

 <p class="word">Озеро по координатам
70°&nbsp;58′&nbsp;19″&nbsp;с.&nbsp;ш.
97°&nbsp;24′&nbsp;5″&nbsp;в.&nbsp;д. 
расположено в Таймырском Долгано-Ненецком 
районе Красноярского края России.</p>

В данном примере для корректного написания координат используется &nbsp;, который не позволяет переносить текст.

Отступы и длина строк

Согласно исследованию Baymard Institute, оптимальная длина строки для удобного чтения — от 50 до 75 символов. Непросто добиться автоматического соответствия этому правилу на любом устройстве. Например, размер шрифта девайса по-умолчанию — 16 px, а мы хотим использовать 20 px. Получается, что наш шрифт должен быть размером 1.25 rem.
