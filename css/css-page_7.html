Вставка спецсимвола - угловые и фигурные скобки

Чтобы каждый раз не вставлять спецсимволы при записи HTML - угловые кавычки, а для CSS - фигурные скобки применим такой ход: пропишем для тега или свойству свой класс.

.st { quotes: "<" ">";}
.st:before {content: open-quote;}
.st:after {content: close-quote;}

.ss { quotes: "{" "}";}
.ss:before {content: open-quote;}
.ss:after {content: close-quote;}
