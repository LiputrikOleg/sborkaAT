Позиция маркера

Для задания позиции маркера, мы можем использовать свойство list-style-position

Для задания маркера, как картинки, мы можем использовать свойство list-style-image

Примеры позиционирования маркеров-картинок:

  • style="list-style-position: inside ;" - Это значение указывает что маркер будет использоваться внутри нашего списка. И будет являться частью текста.
  • style="list-style-position: outside ;" - Это значение указывает, что маркер будет использоваться снаружи нашего списка. И уже не будет являться его частью.
  • style="list-style-image: url('../../images/star_full.png');" - Вставка картинки.

Вложенные списки, другие варианты стилизации смотри на этом сайте

hidden - Имеет тот же эффект, что и none за исключением применения border-style к ячейкам таблицы, у которой значение свойства border-collapse установлено как collapse. В этом случае вокруг ячейки граница не будет отображаться вообще.

Разрешается использовать одно, два, три или четыре значения, разделяя их между собой пробелом. Эффект зависит от количества и указан ниже:

  • Число значений 1 - Стиль границы будет задан для всех сторон элемента.
  • Число значений 2 - Первое значение устанавливает стиль верхней и нижней границ, второе — левой и правой.
  • Число значений 3 - Первое значение задаёт стиль верхней границы, второе — одновременно левой и правой границ, а третье — нижней границы.
  • Число значений 4 - Поочередно устанавливается стиль верхней, правой, нижней и левой границ.
