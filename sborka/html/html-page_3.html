Аудио

Тег audio позволяет проигрывать музыкальные композиции на сайте.

Путь к странице можно задать двумя способами.

Через тег sourse , либо через атрибут src 

 <audio src="Путь к аудиофайлу"></audio>
      


  или      

<audio>
  <sourse src="Путь к аудиофайлу" атрибуты>
</audio>

Атрибуты музыкального проигрывателя

  • src - задает путь к аудиофайлу
  • controls - указывает, что неоходимо отобразить элементы контроля плеера.
  • autoplay - задает автоматическое воспроизведение аудифайла. Можно задавать 2 способами , как и controls
  • loop - задает зацикливание воспроизведения. Также можно задавать 2 способами
  • preload - указывает браузеру, как нужно подгружать мультимедиа на странице.
    • none - не загружать файл.
    • auto - стоит по умолчанию, файлы загружаются вместе со страницей.
    • metadata - браузер подгружает только информацию о файле.

Варианты кода

<audio src="media/clip.mp3" controls="controls"></audio>

или

<audio src="media/clip.mp3" controls></audio>


<audio controls="controls">< sourse src="media/clip.mp3" ></audio>

или

<audio controls>< sourse src="media/clip.mp3" ></audio>


<audio controls autoplay>< sourse src="media/clip.mp3" ></audio>

<audio src="media/clip.mp3" controls preload="metadata"></audio>

Видео

Тег video позволяет проигрывать видеофайлы на сайте.

Путь к странице можно задать двумя способами.

Через тег sourse, либо через атрибут src 

 <video src=”Путь к видеофайлу”></video>

или   

<video>
  <sourse src=”Путь к видеофайлу” атрибуты>
</video>

Атрибуты видеоплеера

Атрибуты видеоплеера такие же как и у аудиоплеера, но у него есть несколько своих дополнительных атрибутов.

  • width - указывает ширину проигрывателя в пикселях или процентах.
  • height - указывает высоту проигрывателя в пикселях или процентах. Обычно не указывают, чтобы не нарушать внешний вид проигрывателя.
  • poster - вставляет постер (картинку-заставку) в видео, которая отображается до воспроизведения.

Важно:
Помните, что пропорции постера и видео должны соответствовать , иначе постер будет отображаться не пропорционально и не красиво. Будет искажаться

Примеры написания кода

Вариант кода 1

<video src=”Путь к видеофайлу”></video>
<video>
  <sourse src=”Путь к видеофайлу” атрибуты>
</video>
<video src="media/clip.mp4" controls preload="metadata" poster="media/poster.jpg"></video>

На устаревших браузерах теги <video> и <audio> не работают, поэтому нужно вставлять напоминалку типа "скачать файл" или "обновить браузер", давая соответствующую ссылку.

Вариант кода 2

<video src="media/clip.mp4" controls preload="metadata" poster="media/poster.jpg">
  <p>К сожалению ваш браузер слишком устарел и не поддерживает технологию HTML5 и CSS3.
  Вы не сможите посмотреть видео, но можете его скачать себе на компьютер:
  <a href="media/clip.mp4">скачать видео</a>
  </p>
</vidio>

Тег sourse

Тег sourse - указывает путь к файлу, который необходимо проиграть. Ниже рассмотрим атрибуты этого тега

  • type - указывает тип воспроизводимого файла.
  • src - указывает путь к файлу.
  • media - Для какого устройства должен производиться этот файл.

Атрибут type

Некоторые браузеры не могут воспроизводить медиафайлы определенных расширений, поэтому на сайт необходимо заливать файлы разных расширений и тегах audio и video указывать их типы.

Для перекодировки медиафайлов, и не только, удобно использовать сайт - Онлайн-конвертер: http://www.online-convert.com

Для аудифайлов это mp3 и ogg

Для видеофайлов это mp4, ogg (src="media/clip.ogv" type="video/ogg") и WebM

Пример для музыки, но для видео то же самое

<audio controls loop preload="metadata">
  <source src="media/clip.mp3" type="audio/mpeg">
  <source src="media/clip.ogg" type="audio/ogg">
  <p class="mistake">К сожалению ваш браузер слишком устарел и не поддерживает технологию HTML5 и CSS3. Вы не сможите прослушать музыкальный фрагмент, но можете его скачать себе на компьютер:
    <a href="media/clip.mp4"> скачать аудио</a>
  </p>
</audio>

Атрибут media

Значение атрибута media:

  • all - для всех устройств
  • handheld - для смартфонов, телефонов и иже с ними
  • screen - для экранов мониторов
  • speech - для речевых синтезаторов
  • projection - для проекторов
  • tv - для теликов
  • tty - для телетайпов
  • print - для печати
