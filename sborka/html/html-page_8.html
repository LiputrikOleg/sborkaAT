Font Awesome - 4

Чтобы подключить шрифт Font Awesome на свой веб-сайт вы можете воспользоваться двумя способами:

  1. Самый простой способ – это с помощью Bootstrap CDN. Для этого необходимо добавить одну строчку кода в раздел заголовка сайта. Ничего загружать и устанавливать не надo.

    <link href="//maxcdn.bootstrapcdn.com/font-awesome/4.5.0/css/font-awesome.min.css" rel="stylesheet">

  2. Второй способ заключается в скачивании шрифта Awesome со страницы проекта, его распаковки в каталог сайта и подключения CSS файла "font-awesome.min.css".

    <link rel="stylesheet" href="/css/font-awesome.min.css">

Из архива fontAwesomeBootstrap.zip переносим папки fonts, css, js. В данном варианте кроме обычной начинки bootstrap присутствует css-файл иконочного шрифта, а также сам шрифт. Подключаем как указано выше во втором способе. Также подключаем bootstrap.

Font Awesome - 5

Здесь добавлены иконки в формате SVG

Подключение от внешнего источника (сайта-разработчика)

  1. Eсли хотим использовать SVG формат иконок, добавляем такую строчку:

    <script defer src="https://use.fontawesome.com/releases/v5.3.1/js/all.js"></script>

  2. Если Вы всё же приверженец шрифтовых, то добавляем такую

    <link href="https://use.fontawesome.com/releases/v5.3.1/css/all.css" rel="stylesheet">

Подключение локально

Скачиваем с официального сайта последнюю версию, извлекаем из архива содержимое.

В корне сайта создаем папку fontawesome и загружаем туда:

  1. Для икононок в SVG: файл all.js из каталога fontawesome-free-5.5.0-web --> js --> all.js, либо его минимизированную копию
  2. Для шрифтовых иконок файл all.css из каталога fontawesome-free-5.5.0-web --> css --> all.css, либо его минимизированную копию и папку webfonts.

Соответственно осуществляем подключение

  • Для SVG

    • <script defer src="/fontawesome/all.js"></script>

  • Для шрифта

    • <link href="/fontawesome/all.css" rel="stylesheet">

Работа с иконками

В FontAwesome - 5 подключение иконок немного изменилось относительно 4 версии. Вместо класса fa используются fab, fas вместо fa, far и fas

2 последних и основном в платной версии. В таблице указаны измениения имен иконок в разных версиях .

Еще одно изменение:

/* общий стиль, разделяемый между версиями */
.icon::before {
  display: inline-block;
  font-style: normal;
  font-variant: normal;
  text-rendering: auto;
  -webkit-font-smoothing: antialiased;
}

/* стиль версии 4 (с использованием calendar-o) */
.calendar::before {
  font-family: FontAwesome;
  content: "\f133";
}

/* стиль версии 5 (с использованием обновленного calendar ) */
.calendar::before {
  font-family: "Font Awesome 5 Free";  /* обновленный font-family */
  font-weight: 400; /* regular style/weight , в 5 версии они могут быть различными*/
  content: "\f133";
}

Новое в Font Awesome 5

Благодаря силе SVG в Font Awesome 5 теперь вы можете масштабировать, позиционировать, переворачивать и поворачивать значки произвольно, используя data-fa-transform атрибут element. Вы можете даже комбинировать их для некоторых супер-полезных эффектов.

Маштабирование

Масштабирование влияет на размер значка без изменения или перемещения контейнера. Чтобы масштабировать значки вверх или вниз, используйте grow-# и shrink-# с любым произвольным значением, включая десятичные числа. Единицы 1 / 16em. Для ясности в примере мы добавили цвет фона на значок, чтобы вы могли видеть эффект.

<div class="fa-4x">
  <i class="fas fa-magic" data-fa-transform="shrink-8" ></i>
  <i class="fas fa-magic"></i>
  <i class="fas fa-magic" data-fa-transform="grow-6"></i>
</div>

Позиционирование

Позиционирует расположение значков эффектов без изменения или перемещения контейнера. Чтобы переместить значки вверх, вниз, влево или вправо, использование up-#, down-#, left-# и right-# с любым произвольным значением, в том числе десятичных знаков. Единицы 1 / 16em. 

<<div class="fa-4x">
  <i class="fas fa-magic" data-fa-transform="shrink-8"></i>
  <i class="fas fa-magic" data-fa-transform="shrink-8 up-6"></i>
  <i class="fas fa-magic" data-fa-transform="shrink-8 right-6"></i>
  <i class="fas fa-magic" data-fa-transform="shrink-8 down-6"></i>
  <i class="fas fa-magic" data-fa-transform="shrink-8 left-6" ></i>
</div>

Вращение и отражение

Power Transform поворачивает и переворачивает эффекты иконка и отражение изображения без изменения или перемещения контейнера. Для того, чтобы вращать или флип иконки использовать любую комбинацию rotate-#, flip-v и flip-h с любым произвольным значением. Единицы - это градусы с допустимыми отрицательными номерами (см. Пятый значок в примере). 

<div class="fa-4x">
  <i class="fas fa-magic" data-fa-transform="rotate-90"></i>
  <i class="fas fa-magic" data-fa-transform="rotate-180"></i>
  <i class="fas fa-magic" data-fa-transform="rotate-270"></i>
  <i class="fas fa-magic" data-fa-transform="rotate-30"></i>
  <i class="fas fa-magic" data-fa-transform="rotate--30" ></i>
  <i class="fas fa-magic" data-fa-transform="flip-v"></i>
  <i class="fas fa-magic" data-fa-transform="flip-h"></i>
  <i class="fas fa-magic" data-fa-transform="flip-v flip-h"></i>
</div>

Создание вращающихся иконок

Для создания вращающейся иконки используйте класс fa-spin.
